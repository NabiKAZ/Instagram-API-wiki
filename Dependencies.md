**PHP >= 5.6 is required**

**Dependencies:**

`gd, curl and ffmpeg`

External info. for **Windows** users: http://php.net/manual/en/install.windows.extensions.php

For **OS X** users (using ports):
```
sudo port install php56-gd
sudo port install php56-curl
```

For **Linux** users:
```
sudo apt-get install php5-gd
...
```
